# UI & Design System

**Activation Mode:** Model Decision (UI context) OR included via GEMINI.md

- **Tailwind v4 First:** You must strictly utilize Tailwind CSS v4 token hierarchy and utilities.
- **Composition over Inheritance:** Extract components only when repeated 3+ times. Otherwise, use inline utility classes.
- **Accessibility:** All interactive elements must have proper ARIA attributes, keyboard navigation (tabindex), and sufficient minimum contrast ratios.
