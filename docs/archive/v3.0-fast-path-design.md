# v3.0 Fast-Path / Natural Language Fallback Design

**Legacy Feature:**
In Council v3.0, the `fullstack-council` router included a "Natural Language Classification" fallback. If a user provided an intent without a slash command, the system would attempt to map it to the closest chain.

```markdown
## Fallback — Natural Language Classification
If the user's message contains no slash command, attempt classification once.
Emit: "No command detected. Classifying intent — use /chain-[x] to skip this."
```

**Reason for Deprecation:**
Fuzzy routing was identified as a "silent killer" of deterministic execution. The v3.5 Hardening Refactor removed this fallback in favor of **Mandatory Command Enforcement**, requiring users to use explicit slash commands or the `/resume` primitive to ensure intentional state transitions.
